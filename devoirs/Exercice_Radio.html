Quelle est ton équipe préférée?





Vous devez faire un groupe de bouton avec les valeurs suivantes:
  • Canadiens (value = Montréal)
  • Nordiques (value = Québec)
  • Titan (value = Bathurst)

À la sélection du choix, faites afficher dans le <h3> avec id "affichage" le texte suivant: "Vous avez choisis l'équipe de ????????"
